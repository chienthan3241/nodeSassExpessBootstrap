Music API

The Music API is RESTful API which provides data for different music resources. Authentication is done by sending your API key as "x-mx-reqtoken header". All paths have to be preceded by the base path, e.g. if you want to GET /charts/weekly/latest, you must send the request to https://www.gfk-e.com/music-api/v1/charts/weekly/latest
